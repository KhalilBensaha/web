# Smooth & Modern Web

A beautifully designed, smooth, responsive website built with plain HTML & CSS.

## Features
- Clean hero with gradients
- Responsive grid for features, gallery, and testimonials
- Soft shadows and tasteful animations
- Accessible: semantic markup, skip link, focus styles, prefers-reduced-motion support

## How to run
Just open `index.html` in your browser.

## Customize
- Edit colors and spacing in `css/styles.css` under `:root`
- Replace gallery images (Unsplash links) with local assets if you prefer
- Update text sections directly in `index.html`
